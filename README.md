# Steam-Download-Proxy-Generator
一键生成可从steam下载正版分流的下载器

# 前提
本地已安装对应游戏的最新版本

# 使用说明
启动生成器(`generator.exe`)，选定要分享的游戏，之后会在`output`目录下的对应`{游戏名}`目录内复制本地的游戏清单文件及密钥。
如果里面没有有下载器(`downloader.exe`)就复制进去，默认是有的。
双击下载器即可开始下载正版分流，要分享的话将`{游戏名}`文件夹打包发送即可

一定程度上支持继续上次的下载进度，但是只检测了文件名和文件大小，最好是一次性下载完整
# 注意事项
分享的下载器有没有dlc之类的取决于生成下载器的人有没有购买，如果没购买，分享出去的下载器就不包含dlc文件

# 打包生成器
```shell
uv run build_generator.py
```
# 打包下载器
```shell
uv run build_downloader.py
```
